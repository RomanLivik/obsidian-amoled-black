# AMOLED Black theme for Obsidian

A high-contrast, minimalist theme designed specifically for OLED/AMOLED displays. Experience a deep, distraction-free environment with a focus on modern aesthetics, fluid animations, and premium glassmorphism effects.

<img width="1920" height="1080" alt="start-screen" src="https://github.com/user-attachments/assets/99eab504-114f-4adf-889e-d4d7c72678c7" />


## ✨ Key Features

* True AMOLED Black: Pure black backgrounds to save battery on OLED screens and provide infinite contrast.

* Premium Minimalist Design: Ultra-thin white borders and high-contrast elements for a sharp, sophisticated look.

* Modern UI Aesthetics: Carefully tuned rounded corners (10px - 14px) for a soft yet professional feel.

* Glassmorphism Effects: Subtle backdrop blur on modals, menus, and tooltips for visual depth.

* Fluid Motion: Custom spring-easing animations and interactive hover effects that make the interface feel "alive."

* Focus-Oriented: Muted text colors for long writing sessions, while keeping UI elements crisp and bright.

## 🚀 Installation
From Obsidian (Recommended)

* Open Settings > Appearance.

* Click Browse in the Themes section.

* Search for AMOLED Black.

* Click Install and Use.
(Note: This will be available once the theme is approved in the community gallery).

## Manual Installation

* Download theme.css and manifest.json from this repository.

* In your vault, go to .obsidian/themes/.

*  Create a folder named obsidian-amoled-black.

* Move the files into that folder.

* In Obsidian, go to Settings > Appearance > Themes and select AMOLED Black.

## 🛠 Recommended Settings

For the best experience:

* Font: Inter, JetBrains Mono, or your favorite system font.

* Translucency: Enabled (for enhanced glass effects).
